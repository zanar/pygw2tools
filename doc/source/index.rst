Welcome to pyGw2Tools's documentation!
======================================

This project is a set of off-game tools for `GuildWars 2 <https://heartofthorns.guildwars2.com/>`_, based on the `WebAPI <https://github.com/arenanet/api-cdi>`_ provided by ArenaNet (`doc here <https://wiki.guildwars2.com/wiki/API:2>`_).

This documentation is a minimal description of the mains classes created for this project.
